# PhD Thesis

This repository contains a University of Pittsburgh ETD LaTeX thesis project.

## Overleaf

Import this repository from GitHub and set the main document to:

```text
main.tex
```

The current project uses the Pitt `pittetd` class in PhD dissertation mode. The first chapter contains a draft introduction based on the distributed coherent optical computing paper and the remaining chapters are placeholders for later expansion.

## Local Structure

- `main.tex`: dissertation entry point
- `chapters/`: chapter files
- `frontmatter/`: abstract and preface placeholders
- `images/`: selected thesis figures copied from the first paper
- `references.bib`: starter bibliography
