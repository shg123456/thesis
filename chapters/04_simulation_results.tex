\chapter{Fiber Experimental Validation}

\section{Experimental Objective}

The fiber experiment tests whether the injection-locking behavior predicted by the rate-equation simulations appears in a physical optical link. The key comparison in this chapter uses an experimental injection ratio of -15~dB and compares it with the -40~dB simulation condition from Chapter~3. The difference between these numerical values reflects the parameter matching between the experimental laser powers and the normalized simulation model; throughout this comparison the simulation uses $\alpha=2$ and threshold 1.944, corresponding to the injected-laser output near 0~dBm.

The measurements are organized as a sequence of increasingly system-relevant tests. First, an unmodulated source laser is used to measure the baseline optical injection-locking response and to identify the locking boundary as the laser detuning is swept. Second, an electro-optic modulator is inserted in the source path so that the remote carrier carries sidebands before injection. This modulated-source case tests whether the locked laser behaves as a clean carrier recovery element or whether source modulation is transferred into the local laser output. Finally, the locked carrier is used in a coherent fiber interference setup, which connects the laser-level locking measurement to the optical multiplication mechanism required for distributed coherent processing.

\section{Unmodulated Source Injection}

The unmodulated experiment measures the baseline carrier-locking behavior. A tunable source laser is injected into the target laser through a fiber-based injection path, and the injected laser output is monitored while the detuning is swept. This measurement establishes the locking range and identifies the residual RF features that remain after the carrier is locked.

Figure~\ref{fig:fiber_unmodulated_setup} shows the unmodulated injection-locking setup. The source laser provides the master optical field. A polarization controller (PC1) aligns the input polarization, and a variable optical attenuator (VOA) sets the injection ratio before the signal enters the optical circulator. The injected laser is connected to another port of the circulator through PC2, which controls the polarization of the injected field at the laser input. The circulator routes the injection signal into the laser while directing the laser output toward the measurement path. PC3 is used to optimize the output polarization before a 50/50 splitter sends the signal to the optical spectrum analyzer (OSA) and to a photodetector (PD). The OSA records the optical spectrum during the detuning sweep, while the PD and spectrum/vector analyzer (SVA) measure the RF response of residual intensity oscillations.

\section{Modulated Source Injection}

The modulated-source experiment adds data-like sidebands to the source before injection. This configuration is closer to the intended distributed coherent computing system, where the remote optical carrier also carries information. The measurement probes whether the injected laser locks primarily to the carrier or whether source modulation leaks into the recovered local carrier.

The modulated setup in Fig.~\ref{fig:fiber_modulated_setup} keeps the same injection and monitoring path as the unmodulated measurement, but adds an electro-optic modulator (EOM1) after the source laser. Waveform generator WG1 drives the modulator and sets the modulation format and depth. PC1 before the EOM adjusts the source polarization for efficient modulation, and the VOA after the modulated path again controls the injection ratio. After the circulator, PC3, splitter, PD, OSA, and SVA play the same roles as in the unmodulated experiment. This matched layout is important because changes between the two measurements can then be attributed primarily to the presence of source modulation rather than to a different optical path.

Figure~\ref{fig:fiber_experiment_neg15_summary} compares the unmodulated and modulated fiber measurements with the matched simulation results. The experimental -15~dB condition is not treated as numerically identical to the simulated -40~dB condition; instead, the simulation is fitted through the calibrated model parameters $\alpha=2$ and threshold 1.944, which correspond to the injected-laser operating point near 0~dBm. Under this matching, the unmodulated experiment and simulation show the same qualitative structure: a detuning-dependent locking boundary, strong residual oscillation features outside the stable region, and a lower-response region near the locked carrier. In the modulated case, the reference-subtracted experimental maps isolate the sideband-induced response from the background source-laser contribution, showing additional RF and spectral features created by the injected modulation. The AM-to-AM simulation reproduces the same trend by showing strong modulation transfer near sensitive detuning branches and weaker transfer in the stable locked region. This agreement supports the use of the rate-equation model as a design tool for selecting operating points that preserve carrier locking while suppressing unwanted modulation transfer.

\section{Comparison with the Simulation Model}

The experimental -15~dB injection condition is compared with the -40~dB simulation condition using $\alpha=2$ and threshold 1.944. The threshold value is chosen because it matches the injected-laser output condition near 0~dBm. The comparison is therefore qualitative and model-matched rather than a direct equality between optical power ratios. In both experiment and simulation, the useful operating region is the one where the carrier remains locked while the residual RF response and sideband transfer remain low.

The threshold comparisons in Figs.~\ref{fig:threshold_115_locking_region} and~\ref{fig:threshold_194_locking_region} also explain why this matching is important. Changing the threshold changes the simulated locking boundary and residual response pattern, so the same nominal injection ratio can represent different physical operating regimes unless the threshold is matched to the laser output condition.

\section{Coherent Fiber Interference Setup}

After carrier locking is established, the recovered local carrier can be remodulated and interfered with a retained branch of the remote signal. The fiber interference setup is the experimental bridge between laser-level locking measurements and coherent optical computation. It uses the injection-locked laser as the local coherent source and balanced detection to extract the interference term.

In this interference experiment, phase drift in the fiber paths must be actively controlled. The retained remote branch and the remodulated local branch interfere at the output splitter, and the detected interference signal depends on their relative optical phase. Environmental drift, fiber motion, and thermal fluctuations can move the interferometer away from the desired operating point. A PID feedback loop is therefore used to stabilize the phase by driving a fiber stretcher in one arm of the interferometer.

This feedback suppresses slow phase drift and keeps the interference output near a controlled operating point, which is necessary for repeatable coherent multiplication and balanced-detection measurements.

\section{Summary}

The fiber measurements support the simulation-based design picture. Optical injection locking can provide a coherent local carrier, but the operating point must be selected using both carrier locking and residual modulation transfer. The -15~dB fiber measurements, interpreted through the $\alpha=2$, threshold 1.944 simulation model, show that modulation can modify the RF response even when the carrier-level locking signature remains visible. The next stage is to translate this fiber architecture into chip-scale and distributed system layouts.
